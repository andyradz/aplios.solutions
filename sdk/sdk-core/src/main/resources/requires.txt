SPACJA » Wybierz plik lub katalog (jak INSERT). Po wybraniu niezaznaczonego katalogu wskazanego kursorem przy użyciu klawisza SPACJA, określona zostanie wielkość znajdujących się w nim plików i podkatalogów, która następnie zostanie wyświetlona, w pełnym widoku, zamiast ciągu <DIR>. Zachowanie to można zmienić przy użyciu opcji 'Konfiguracja' - 'Opcje' - 'Działania' - 'Przy wybieraniu spacją'.
ENTER » Zmień katalog / uruchom program / uruchom skojarzony program / wykonaj wiersz poleceń, jeśli zawiera polecenia. Jeśli panel źródłowy wyświetla zawartość archiwum, zostaną wyświetlone jego szczegółowe dane.
SHIFT+ENTER »
1. Uruchom wiersz poleceń / program wskazany kursorem, poprzedzając jego nazwę poleceniem cmd /c i pozostawiając okno programu otwarte po jego zakończeniu. Funkcja ta działa, jeśli w katalogu Windows znajduje się plik NOCLOSE.PIF!
2. Dla plików ZIP: Użyj opcji alternatywnej do określonej na stronie Kompresory: (Obsługuj archiwa jak katalogi <-> wywołaj skojarzony program, tzn. winzip lub quinzip).
3. Wewnątrz archiwum: Rozpakuj plik wskazany kursorem i traktuj go jak archiwum (przetwarzanie zagnieżdżonych archiwów zip).
4. W liście ostatnio używanych katalogów (Historia, Ctrl+D) otwiera katalog w nowej zakładce.
